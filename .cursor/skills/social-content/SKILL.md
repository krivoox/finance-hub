# Social Content Skill

## Purpose
Generate LinkedIn and X content from repository progress.

Always use the `/linkedin-posts` skill as the source of truth for tone, structure and writing style.

## Process
- Analyze repository.
- Detect meaningful changes.
- Choose the strongest educational angle.
- Avoid promotional language.
- Teach something.

## Deliverables
1. Summary
2. Why it matters
3. LinkedIn post
4. X thread (4-8 tweets)
5. Hashtags
6. Image suggestions
7. Three future content ideas

If nothing is worth publishing, return:
`No publication worth posting today.`
